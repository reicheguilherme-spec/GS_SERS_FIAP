## Descrição
Sistema desenvolvido para monitoramento de uma missão espacial experimental.

O sistema analisa:
- Temperatura
- Energia
- Comunicação
- Status dos módulos

Também gera alertas automáticos e toma decisões básicas em situações críticas.
